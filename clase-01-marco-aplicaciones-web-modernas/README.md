# Clase 01 - Marco de aplicaciones web modernas

Esta carpeta contiene la primera clase del curso Desarrollo Web II con enfoque en FastAPI.

## Estructura

- `clase-01.md`: documento principal de la clase, con explicacion amplia, ejemplos, diagnostico, mini-laboratorio y bibliografia.
- `html/index.html`: diapositivas de apoyo para proyectar en clase.
- `css/slides.css`: estilos de las diapositivas.
- `js/slides.js`: navegacion de la presentacion.
- `dudas/preguntas-frecuentes.md`: banco inicial de preguntas y aclaraciones.
- `ejercicios/ejercicios-clase-01.md`: ejercicios de practica y reflexion.
- `ejemplo-guiado/consumir-api-publica-python.md`: paso a paso para consumir una API publica desde Python.

## Uso sugerido

1. Leer `clase-01.md` como guia docente principal.
2. Abrir `html/index.html` en el navegador para usar las diapositivas.
3. Aplicar la actividad diagnostica durante la clase.
4. Asignar los ejercicios como practica o trabajo independiente.
5. Usar el ejemplo guiado para conectar HTTP, APIs y Python antes de empezar a construir con FastAPI.
