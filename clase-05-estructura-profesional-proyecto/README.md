# Clase 05 - Estructura profesional de un proyecto FastAPI

Esta carpeta contiene la quinta clase del curso Desarrollo Web II con enfoque en FastAPI.

## Estructura

- `clase-05.md`: documento principal de la clase.
- `html/index.html`: diapositivas de apoyo.
- `css/slides.css`: estilos de las diapositivas.
- `js/slides.js`: navegacion de la presentacion.
- `dudas/preguntas-frecuentes.md`: preguntas frecuentes.
- `ejercicios/ejercicios-clase-05.md`: ejercicios practicos.
- `ejemplo-guiado/proyecto-modular-fastapi.md`: guia paso a paso.
- `ejemplo-guiado/proyecto-modular/`: proyecto FastAPI modular ejecutable.

## Uso sugerido

1. Leer `clase-05.md` como guia docente principal.
2. Usar `html/index.html` como apoyo visual.
3. Ejecutar el refactor modular durante clase.
4. Usar `ejemplo-guiado/proyecto-modular/` como referencia.
5. Asignar el README tecnico y endpoints de salud/version como evidencia.
