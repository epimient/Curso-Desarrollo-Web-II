Entorno virtual

¿Que es un entorno virtual?

En español simple Es una caja aislada donde instalas las librerias de tu proyecto sin contaminar el resto del sistema. Cada proyecto tiene su propia caja.
# Crear la caja
python -m venv venv

# Activar la caja (Linux/Mac)
source venv/bin/activate

# Activar la caja (Windows)
venv\Scripts\activate

Veras (venv) al inicio de la terminal. Eso indica que estas dentro de la caja.

pip

pip: el mensajero de paquetes

Tu proyecto pip install PyPI 
pip install fastapi
pip install "uvicorn[standard]"

# O en un solo comando:
pip install fastapi uvicorn[standard]

PyPI es el repositorio oficial de paquetes Python. pip descarga e instala lo que pidas.

Hola mundo

main.py: tu primer endpoint

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def raiz():
    return {"mensaje": "Hola FastAPI!"}

Guardalo como main.py. Esto es todo lo que necesitas para tener una API funcionando.

Uvicorn

Uvicorn: el motor que enciende la API

En español simple FastAPI construye la API (el auto), Uvicorn es el motor que la hace andar. Sin Uvicorn, tu codigo no escucha peticiones.
uvicorn main:app --reload
ParteSignificado
mainNombre del archivo (sin .py)
appVariable FastAPI dentro del archivo
--reloadReinicia automaticamente al cambiar codigo

Documentacion

/docs y /redoc: documentacion automatica

FastAPI genera documentacion interactiva sin esfuerzo:

http://localhost:8000/docs   # Swagger UI
http://localhost:8000/redoc  # ReDoc

Puedes probar los endpoints directamente desde el navegador. Escribe datos, ejecuta y ve la respuesta.

Ventaja competitiva Mientras otros frameworks requieren librerias extra para documentar, FastAPI la genera automaticamente con OpenAPI.

Error 422

Error 422: validacion fallida

@app.post("/cursos")
def crear_curso(curso: Curso): ...

# Si envias:
# { "nombre": "Web" }
# Falta "creditos" → 422 Unprocessable Entity

FastAPI valida automaticamente que los datos coincidan con el modelo. Si algo falta o es incorrecto, devuelve 422 con el detalle.

Asi NO vs Asi SI

Errores de instalacion y ejecucion

❌ Ejecutar sin activar venv

# Instalaste en venv pero ejecutas fuera
python main.py  # ModuleNotFoundError

El modulo se instalo dentro del venv, pero estas fuera.

✅ Activar venv primero

source venv/bin/activate
(venv) $ uvicorn main:app --reload

Siempre verifica que veas (venv) en la terminal.

Actividad

Tu primera API con dos endpoints

  1. Crea un entorno virtual e instala FastAPI + Uvicorn.
  2. Crea un endpoint GET /saludo que reciba un query param nombre.
  3. Crea un endpoint GET /usuarios/{id} que devuelva datos mock.
  4. Prueba ambos endpoints en /docs.
  5. Agrega validacion: si el id es menor a 1, devuelve error 422.

Cierre

Lo que debe quedar claro

  • Entorno virtual = caja aislada por proyecto.
  • Uvicorn es el servidor ASGI que ejecuta FastAPI.
  • Los decoradores @app.get() definen rutas.
  • FastAPI convierte automaticamente tu respuesta a JSON.
  • /docs es tu mejor aliado para probar la API.
