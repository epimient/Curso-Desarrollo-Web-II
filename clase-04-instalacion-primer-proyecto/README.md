# Clase 04 - Instalacion, entorno y primer proyecto FastAPI

Esta carpeta contiene la cuarta clase del curso Desarrollo Web II con enfoque en FastAPI.

## Estructura

- `clase-04.md`: documento principal de la clase, con explicacion amplia, comandos, endpoints, errores comunes y bibliografia.
- `html/index.html`: diapositivas de apoyo para proyectar en clase.
- `css/slides.css`: estilos de las diapositivas.
- `js/slides.js`: navegacion de la presentacion.
- `dudas/preguntas-frecuentes.md`: banco de preguntas y aclaraciones.
- `ejercicios/ejercicios-clase-04.md`: ejercicios de instalacion, endpoints y documentacion.
- `ejemplo-guiado/primer-proyecto-fastapi.md`: guia paso a paso del primer proyecto.
- `ejemplo-guiado/proyecto-base/`: proyecto FastAPI minimo ejecutable.

## Uso sugerido

1. Leer `clase-04.md` como guia docente principal.
2. Abrir `html/index.html` en el navegador para usar las diapositivas.
3. Desarrollar el taller guiado en vivo.
4. Usar `ejemplo-guiado/proyecto-base/` como referencia o punto de partida.
5. Asignar los ejercicios como trabajo independiente.
