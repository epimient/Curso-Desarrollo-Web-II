# Clase 03 - Frameworks web: comparacion y eleccion tecnica

Esta carpeta contiene la tercera clase del curso Desarrollo Web II con enfoque en FastAPI.

## Estructura

- `clase-03.md`: documento principal de la clase, con explicacion amplia, comparacion de frameworks, matriz, actividad ABP y bibliografia.
- `html/index.html`: diapositivas de apoyo para proyectar en clase.
- `css/slides.css`: estilos de las diapositivas.
- `js/slides.js`: navegacion de la presentacion.
- `dudas/preguntas-frecuentes.md`: banco de preguntas y aclaraciones.
- `ejercicios/ejercicios-clase-03.md`: ejercicios de comparacion, matriz y argumentacion tecnica.
- `ejemplo-guiado/matriz-decision-frameworks.md`: paso a paso para construir una matriz de decision.

## Uso sugerido

1. Leer `clase-03.md` como guia docente principal.
2. Abrir `html/index.html` en el navegador para usar las diapositivas.
3. Dividir el grupo en equipos para resolver los casos ABP.
4. Solicitar una matriz comparativa como evidencia.
5. Cerrar con la justificacion de FastAPI como framework central del curso.
