# Clase 02 - MVC, patrones y equivalencia arquitectonica en FastAPI

Esta carpeta contiene la segunda clase del curso Desarrollo Web II con enfoque en FastAPI.

## Estructura

- `clase-02.md`: documento principal de la clase, con explicacion amplia, ejemplos, actividad y bibliografia.
- `html/index.html`: diapositivas de apoyo para proyectar en clase.
- `css/slides.css`: estilos de las diapositivas.
- `js/slides.js`: navegacion de la presentacion.
- `dudas/preguntas-frecuentes.md`: banco de preguntas y aclaraciones.
- `ejercicios/ejercicios-clase-02.md`: ejercicios de practica, analisis y diseno.
- `ejemplo-guiado/api-cursos-por-capas.md`: paso a paso para construir una API de cursos separada en routers, schemas y services.

## Uso sugerido

1. Leer `clase-02.md` como guia docente principal.
2. Abrir `html/index.html` en el navegador para usar las diapositivas.
3. Desarrollar la actividad de desarmar un CRUD en capas.
4. Asignar el diagrama arquitectonico como evidencia semanal.
5. Usar el ejemplo guiado para preparar la transicion hacia instalacion y primer proyecto FastAPI.
